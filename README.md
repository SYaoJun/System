# 解析CSV文件
## 目标
- 掌握getopt的用法
- 掌握fgets读取文件
- 使用malloc分配内存
## 命令行参数
```c
Usage: ./select -c NCOLS [-n NRECS] COL1 COL2 COL3 ...
```

`./select`：可执行程序的名称。

`-c NCOLS`：要从输入 CSV 中读取的列数。

`[-n NRECS]`：可选。要解析的最大行数。

`COL1 COL2 COL3 ...`：要输出的列的索引（索引从1开始）。

## Task 1: 解析命令行参数
- 第一行是状态栏。
- 列的索引从1开始编号。
- 列可以重复。
- n是最大行数。
```c
// 读取4行文件，输出第1列和第4列
./main -c 4 1 4 < lotr.txt
```

## Task 2: 处理文件输入
- 文件格式是正确。
- 文件可能不存在。
- 文件存在可能权限不足。
